Introduction

Welcome!

Welcome to the Cyber Range! This will serve as a place to practice your offensive (red team) skills. The range contains a variety of machines that you can hack into. There are machines for everyone from easy to insane and both Linux and Windows!

Below are a few places on the range that you should check out:

  1. The Scoreboard - An up-to-date ranking of everyone who has logged into the Cyber Range!
  2. Your Profile - This will be updated with a timeline of your "owns" as you progress!
  3. The Machines - A list of machines you can try and hack!

Machine Introduction

On the machines page, you will see every machine that is currently deployed on the Cyber Range.

At the top of the card you will see two things: the bold text indicating the machine's name and a colored circle indicating the machine's status.

Inside the card from top to bottom you will see an icon (either Tux the Linux penguin or the Windows logo) denoting the type of machine, the machine's IP address if it is currently active, and the number of user ($) and root (#) owns.

At the bottom of the card you will see four buttons, "Start", "Stop", "Revert", and "Flags".

  1. Start, stop, and revert all perform actions on the machine according to their name. Reverting a machine resets it to its initial state - this is useful if someone deletes a flag or makes the machine unusable.
  2. The "Flags" button is for submitting the user and root flags of the machine - these will typically be found at /home/username/user.txt and /root/root.txt for Linux machines and C:\Users\Username\Desktop\user.txt and C:\Users\Administrator\Desktop\root.txt for Windows machines.
