tar

The tar command is very similar to the zip command, in which it allows us to collect many files into a single archive file, referred to as a tarball. Unfortunately, Linux does not come with a untar command, so the tar command is used for both creating the archive and extracting it. To switch between making a tar file and extracting one, we use flags. To learn what each flag does, make sure to take advantage of the man pages. In our case, we simply want to extract the tarball, so we can use:

tar -xvf fileName.tar

  • -x: Extract the tarball
  • -v: Verbose output and show the progress of extraction
  • -f: specify the name of the tar file (fileName.tar)
