Linux Filesystem Hierarchy

The starting directory of all Linux machines is the /, or the root directory.This directory contains all the underlying files and directories for the machine. There are some common Linux directories that usually always exist, but depending on the distribution of Linux, they might not.

Here is a diagram of all the standard Linux file system layout.

Some of the important directories include:

  • bin: common programs shared by the system
  • boot: startup files and the kernel(the actual code of the operating system)
  • etc: contains important system configuration files
  • home: home directories for users
  • lib: library files, needed for all kinds of programs
  • opt: contains extra or third-party software
  • tmp: temporary space for use of the system, cleaned up on reboot of the system
  • usr: programs, libraries, documentation for all user-related programs
  • var: storage for all variable files and temporary files created by users
