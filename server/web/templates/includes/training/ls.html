ls

ls or the "list" command is one of the simplest but most commonly used commands while using the Linux terminal. This command simply lists the files and directories within the current directory.Think of directories as folders that you can put files into, for instance, if you have a folder on your Desktop of your personal computer, that folder is a directory within the Desktop directory. There are some specific directories that come with Linux that we will talk more about later on.

Try running the ls command in your terminal.
