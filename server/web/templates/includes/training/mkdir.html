mkdir

This is a command that you may use called mkdir, which stands for "make directory", and is used for exactly that. It takes an argument of the path that you would like to make the directory in. For example, to make a directory in your current working directory, you would use:

mkdir nameOfDirectory

Or, to make a directory in some place other than your current working directory you would use:

mkdir /path/to/directory/nameOfDirectory

Try creating a directory in your current working directory called flag_four. The flag for this section will be the path of flag_four directory.
