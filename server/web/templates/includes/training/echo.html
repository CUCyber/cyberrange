echo

The echo command does exactly what it sounds like. If you send a string as the argument for echo, it will output that string back out to standard output or your screen. This is incredibly convenient for our base64 situation because we can use it to encode or decode strings. For example, if we wanted to base64 encode the string "Hello World!1!!1!", we would use:

echo "Hello World!1!!1!" | base64
