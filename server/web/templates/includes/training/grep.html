grep

Grep is a super useful command that will come in handy if you ever want to search a file or multiple files for something. The grep command will return whatever it found the search phrase or nothing if the search came up empty. The grep syntax goes as follows:

grep "phrase you want" fileName

Use this grep command to find the flag within the flag_fourteen file.
